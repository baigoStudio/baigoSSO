安装方法

1、将下载到的程序包解压，然后将所有文件上传到服务器，假设网站 URL 为 http://www.domain.com 上传到根目录 /，以下说明均以此为例；
2、用浏览器打开 http://www.domain.com/install/ 按提示操作。

---------------------------------------------------------------------

升级方法

1、升级前请务必 备份数据库 如果您在最初部署 baigo SSO 时，采用了特殊的部署方法，修改了 ./config/config.inc.php 文件，请同时备份该文件；
2、将下载到的程序包解压，然后将所有文件上传到服务器，假设网站 URL 为 http://www.domain.com 上传到根目录 /，以下说明均以此为例；
4、登录后台，即用浏览器打开 http://www.domain.com/admin/ 系统将自动跳转到升级界面。或者直接用浏览器打开 http://www.domain.com/install/ctl.php?mod=upgrade 按提示操作。
