数据库设置

接口说明

本接口用于设置数据库，本接口必须在第一步调用。

URL

http://www.domain.com/api/api.php?mod=setup

HTTP 请求方式

POST

返回格式

JSON

返回结果是否加密

否

请求参数
名称 类型 必须 描述
act string true 接口调用动作，值只能为 dbconfig。
db_host string true 数据库服务器
db_port string true 服务器端口
db_name string true 数据库名称
db_user string true 数据库用户名
db_pass string true 数据库密码
db_charset string true 数据库字符编码
db_table string true 数据表前缀
 
返回结果
名称 类型 描述
rcode string 返回代码，详情查看 返回代码。
prd_sso_ver string baigo SSO 版本号。
prd_sso_pub string baigo SSO 版本发布时间，格式为年月日。
 

返回结果示例

{
    "prd_sso_ver": "1.1.1", //SSO 版本号
    "prd_sso_pub": 20150923, //SSO 版本发布时间
    "rcode": "y010102" //返回代码
}

基本设置

接口说明

本接口用于基本设置。

URL

http://www.domain.com/api/api.php?mod=setup

HTTP 请求方式

POST

返回格式

JSON

返回结果是否加密

否

请求参数
名称 类型 必须 描述
act string true 接口调用动作，值只能为 base。
opt array true 安装参数，详情查看示例。
 

安装参数

array(
    "opt" => array(
        "base" => array(
            "BG_SITE_NAME"        => "baigo SSO" //站点名称
            "BG_SITE_DOMAIN"      => "www.domain.com" //域名
            "BG_SITE_URL"         => "http://www.domain.com" //首页 URL
            "BG_SITE_PERPAGE"     => 30 //每页显示数
            "BG_SITE_TIMEZONE"    => "Asia/Shanghai" //时区
            "BG_SITE_DATE"        => "Y-m-d" //日期格式，等同于 PHP date 函数
            "BG_SITE_DATESHORT"   => "m-d" //短日期格式，等同于 PHP date 函数
            "BG_SITE_TIME"        => "H:i:s" //时间格式，等同于 PHP date 函数
            "BG_SITE_TIMESHORT"   => "H:s" //短时间格式，等同于 PHP date 函数
            "BG_SITE_TPL"         => "default" //模板
            "BG_ACCESS_EXPIRE"    => 60; //访问口令存活期，单位：分钟
            "BG_REFRESH_EXPIRE"   => 60; //刷新口令存活期，单位：天
            "BG_VERIFY_EXPIRE"    => 30 //验证链接有效期，单位：分钟
        )
    )
);

 
返回结果
名称 类型 描述
rcode string 返回代码，详情查看 返回代码。
prd_sso_ver string baigo SSO 版本号。
prd_sso_pub string baigo SSO 版本发布时间，格式为年月日。
 

返回结果示例

{
    "prd_sso_ver": "1.1.1", //SSO 版本号
    "prd_sso_pub": 20150923, //SSO 版本发布时间
    "rcode": "y010102" //返回代码
}

URL

http://www.domain.com/api/api.php?mod=setup

POST

JSON

完成安装

接口说明

本接口用于通知系统安装已完成，本接口必须在最后一步调用，本接口调用成功后，安装接口将全部失效。

URL

http://www.domain.com/api/api.php?mod=setup

HTTP 请求方式

POST

返回格式

JSON

返回结果是否加密

否

请求参数
名称 类型 必须 描述
act string true 接口调用动作，值只能为 over。
app_name string true 应用名称。
app_url_notify string true 通知 URL。
app_url_sync string true 同步 URL。
 
返回结果
名称 类型 描述
rcode string 返回代码，详情查看 返回代码。
sso_url string SSO API 接口的 URL。
app_id int 调用 API 接口所需的 APP ID。
app_key string 调用 API 接口所需的 APP KEY。
prd_sso_ver string baigo SSO 版本号。
prd_sso_pub string baigo SSO 版本发布时间，格式为年月日。
 

返回结果示例

{
    "prd_sso_ver": "1.1.1", //SSO 版本号
    "prd_sso_pub": 20150923, //SSO 版本发布时间
    "app_id": "MTA=", //应用 APP ID
    "app_key": "sfewrw8084382h2r9fdsw9ey5whfDISORwegds", //应用 APP KEY
    "rcode": "y010102" //返回代码
}
