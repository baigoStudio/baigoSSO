更换邮箱

文件名：profile_mailbox.php

用户更换邮箱找回后将会收到确认邮件，确认邮件将用户带至本界面。

键名 说明 备注
$this->tplData['userRow'] 用户详细信息  
$this->tplData['verifyRow'] 验证详细信息 其中 $this->tplData['verifyRow']['verify_mail'] 为新邮箱
 

$this->tplData['userRow'] 数组

当前用户详细信息

键名 类型 说明 备注
user_id int 用户 ID  
user_name string 用户名  
user_mail string 邮箱  
user_contact array 其他联系方式  
user_extend array 扩展信息  
user_nick string 昵称  
user_status string 用户状态 enable 为激活，wait 为待激活，disable 为禁用。
rcode string 返回代码 显示当前用户的状态，详情请查看 返回代码。
 
用户数据示例
print_r($this->tplData['userRow']) 

Array (
    [user_id] => 2 //ID
    [user_name] => user //名称
    [rcode] => y010102 //返回代码
)
