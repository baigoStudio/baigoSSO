公共请求参数

公共请求参数是指向所有接口发起请求时都必须传入的参数。

名称 类型 必须 描述
app_id int true 应用的 APP ID，后台创建应用时生成的 ID。详情查看 应用。
app_key string true 应用的 APP KEY，后台创建应用时生成的 KEY。详情查看 应用。
time int true Unix 时间戳，baigo SSO 允许 +-30 分钟以内的时差，为了防止时区设置不同导致的时差，请开发者将应用的时区设置为为与 baigo SSO 一致，关于时区设置，请查看 系统设置。
sign string true 签名，将请求参数中的加密参数的值用 Base64 编码（注：切勿将参数名进行 Base64 编码），然后拼合成 JSON 对象，加以签名，签名方式请查看 签名。

发送短信

接口说明

本接口用于向指定用户发送站内短信。

URL

http://www.domain.com/api/api.php?m=pm&c=api

HTTP 请求方式

POST

返回格式

JSON

返回结果是否加密

否

 

请求参数

名称 类型 必须 描述
a string true 接口调用动作，值只能为 send。
code string true 加密参数，将加密参数的值用 Base64 编码（注：切勿将参数名进行 Base64 编码），然后拼合成 JSON 对象，最后用加密函数加密，加密方法请查看 密文。
 

加密参数

将下列参数的值用 Base64 编码（注：切勿将参数名进行 Base64 编码），然后拼合成 JSON 对象，最后用加密函数加密，加密方法请查看 密文。

名称 类型 必须 描述
user_id int true

三选一，优先级为 user_id > user_name > user_mail（作为发送者）

其中是否允许邮箱登录，视注册设置情况而定。详情查看 注册设置。
user_name string
user_mail string
user_access_token string true 访问口令，必须用 MD5 加密后传输。
pm_to string true 接收者用户名，多个接收者请用 | 分隔。
pm_title string false 短信标题
pm_content string true 短信内容
 

返回结果

名称 类型 描述
rcode string 返回代码，详情查看 返回代码。
 
返回结果示例

{
    "rcode": "y110101" //返回代码
}

列出短信

接口说明

本接口用于列出当前用户的所有短信。

URL

http://www.domain.com/api/api.php?m=pm&c=api

HTTP 请求方式

GET

返回格式

JSON

返回结果是否加密

是

 

请求参数

名称 类型 必须 描述
a string true 接口调用动作，值只能为 list。
code string true 加密参数，将加密参数的值用 Base64 编码（注：切勿将参数名进行 Base64 编码），然后拼合成 JSON 对象，最后用加密函数加密，加密方法请查看 密文。
 

加密参数

将下列参数的值用 Base64 编码（注：切勿将参数名进行 Base64 编码），然后拼合成 JSON 对象，最后用加密函数加密，加密方法请查看 密文。

名称 类型 必须 描述
user_id int true

三选一，优先级为 user_id > user_name > user_mail（作为接收者）

其中是否允许邮箱登录，视注册设置情况而定。详情查看 注册设置。

pm_status string false 短信状态，wait（未读）、read（已读）。
pm_ids string false 短信 ID，多个 ID 请用 | 分隔。
per_page int false 每页列出短信数。
key string false 关键词。
 

返回结果

本接口返回加密结果，真正内容需要解密。详情查看 密文。

本接口返回加密结果，真正内容需要解密。详情查看 密文。

        "user_name": "MTA="
    },
    "toUser": { //接收者用户信息
        "user_id": "MTA="
        "user_name": "MTA="
    },
